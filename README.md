# Data Discovery & Visualisation assignment
## Context:
This is an academic project on data visualisations and dashboards for a fitness company. The project aims to identify patterns between customer purchasing behaviour and product characteristics, so that sales managers are incentivised and can act to increase sales.

## File(s):
- Data Visualisation.ipynb holds the code for the dashboards and data visuals. The report resides in the same file.
- cardio_good_fitness.csv holds the sales data of the company (used in the .ipynb file).

## Note(s):
All documentation, such as steps taken and reasoning, is in the .ipynb file.
